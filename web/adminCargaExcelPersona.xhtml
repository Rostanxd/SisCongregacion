Para poder realizar la carga del excel de persona, por favor tomar en cuenta lo siguiente:

  • Las columnas que contengan números (celular, teléfono, etc.) deben contener el caracter comilla simple (') al comienzo.
  • Las columnas de Congregación, Grupo y Privilegio, deben tener el id del mismo, mas no el nombre.
  • Para identificar el sexo de la persona utilizar "M" para Masculino y "F" para Femenino.
  • Para identificar el estado de la persona utilizar "A" para Activo e "I" para Inactivo.
